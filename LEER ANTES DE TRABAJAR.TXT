Pasos a realizar antes de comenzar con cualquier cosa:
    1) recuerden hacer un pull del proyecto antes de siquiera salir de la rama main
        "git pull origin main"
    2) en Trello, coloquen específicamente sobre que componente están trabajando, así sabemos los demás y no tocamos ese archivo
    3) creen una rama específica con el nombre del componente asi {componente}-front
        por ejemplo "git checkout -b footer-front" o "git bash footer-front" y luego "git checkout footer-front"
    4) solo hacer el merge cuando hayan terminado con el componente, o mínimamente esté utilizable por react (no fusionen con main si aún tiene errores al compilar)
        luego del commit "git checkout main" y luego "git merge {componente}-front"

Para poder trabajar con el front, recuerden moverse a la carpeta "cafe-aroma-front" 
 - En la terminal escribir "cd cafe-aroma-front", así estarán dentro del entorno que puede leer React
    (Para instalar dependencias y demas).

Para hacer los commits, deben regresar a la carpeta raiz (front)
 - En la terminal simplemente tipear "cd .." y regresará una carpeta anterior, ahí están dentro del entorno de git
    (Para crear ramas, commitear y fusionar ramas y demas)
